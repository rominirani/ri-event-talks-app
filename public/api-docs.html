HelloWorld API Docs

RESTful API documentation for the HelloWorld Tech Conference.

GET /api/talks

Fetches the complete schedule of talks for the event. The data returned by this endpoint powers the Dual-Criteria Search (Category & Speaker) and the Interactive Detail View (Modals) on the frontend.

Request Example
fetch('/api/talks')
  .then(response => response.json())
  .then(data => console.log(data));
Response Structure (200 OK)
[
  {
    "id": 1,
    "title": "The Future of Generative AI...",
    "speakers": ["Dr. Sarah Chen", "Marcus Thorne"],
    "categories": ["AI", "Machine Learning"],
    "description": "An in-depth look at LLMs..."
  },
  ...
]
GET / (Static)

Serves the primary web application interface. All files in the /public directory are served as static assets.
